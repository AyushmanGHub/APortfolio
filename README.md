## 📖 About Project

This project showcases a personal portfolio website that integrates an advanced **Agentic AI Assistant (ResAgent)** directly into the user interface, providing an interactive and dynamic experience beyond traditional static portfolio sites.

Typical portfolio websites are limited to showcasing fixed information — projects, certifications, and personal info — but lack interactivity and real-time updates. To address this, the project integrates **LLM-powered query answering, real-time information retrieval, and conversational AI capabilities** into the portfolio, making it highly interactive and personalized for visitors.

The website dynamically presents:

- Personal projects and papers.
- Certifications and courses.
- AI-powered blog integration.
- Conversational AI assistant capable of answering queries, fetching live news, and providing intelligent responses using LLMs.

---

## 📊 Overview

**Streamlit AI Portfolio** leverages modern **Agentic AI** technologies to make personal portfolios smarter, interactive, and responsive.

This project covers:

- ✅ Full-stack **Streamlit** development for responsive UI.
- ✅ Integration of **LangGraph** orchestration for multi-step LLM agents.
- ✅ Backend powered by **Groq LLM APIs** for fast and scalable large language model access.
- ✅ Use of **Retrieval-Augmented Generation (RAG)** to inject personalized, real-time data into the AI assistant.
- ✅ Automated dynamic rendering of certifications, projects, and blogs.
- ✅ Seamless deployment via **Streamlit Cloud**.
